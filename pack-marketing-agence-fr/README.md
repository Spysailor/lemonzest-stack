## 🎯 Qu'est-ce que c'est ?

**Pack Marketing Agence FR** est une suite de **5 skills Claude** conçus spécifiquement pour le marché B2B francophone (France, Belgique, Suisse, Maurice, Afrique francophone).

Là où 95% des skills existants sont en anglais avec une logique US, ce pack respecte :
- ✅ La typographie française (espaces insécables, guillemets, apostrophes)
- ✅ La réglementation (RGPD, LCEN, CNIL, mentions légales)
- ✅ Les codes culturels B2B FR (vouvoiement, ton mesuré, cycles longs)
- ✅ Les outils dominants en FR (Brevo, Matomo, PrestaShop, Kompass)

## 🚀 Installation

### Option 1 — Installation manuelle (Claude Code)

```bash
# 1. Cloner ou télécharger le pack
git clone https://github.com/lemonzest/pack-marketing-agence-fr.git

# 2. Copier dans votre dossier skills
cp -r pack-marketing-agence-fr ~/.claude/skills/

# 3. Vérifier l'installation
ls ~/.claude/skills/pack-marketing-agence-fr/
```

### Option 2 — Installation via plugin marketplace (bientôt)


## 💡 Utilisation

### Activation automatique

Le skill se déclenche quand vous mentionnez :
- "audit SEO"
- "rédiger un article"
- "post LinkedIn pro"
- "séquence email Brevo"
- "proposition commerciale"
- Ou tout contexte francophone B2B / marketing digital

### Exemples de prompts

**Exemple 1 — Audit SEO :**
> "J'ai un site WordPress pour un restaurateur à Lyon, fais-moi un audit SEO rapide"

→ Claude active `seo-audit-fr`, utilise le template rapport, applique contexte FR (Google.fr, Pages Jaunes, SEO local français).

**Exemple 2 — Post LinkedIn :**
> "Écris un post LinkedIn de storytelling sur l'erreur typique que font les PME avec leur SEO"

→ Claude active `linkedin-post-fr`, utilise le format 1 (storytelling), respecte typographie FR, hashtags B2B francophones.

**Exemple 3 — Workflow complet :**
> "Je lance une nouvelle offre SEO, aide-moi à préparer un post LinkedIn de lancement, une séquence email pour mes contacts, et une proposition type"

→ Claude orchestre 3 skills (`linkedin-post-fr` + `email-sequence-fr` + `client-proposal-fr`), avec cohérence de message sur les 3.

## 🎓 Pour qui ?

### Le pack est parfait pour vous si vous êtes :

- **Freelance marketing / SEO / rédaction** en contexte francophone
- **Agence digitale** qui sert des clients FR/BE/CH/MU
- **Consultant indépendant** qui vend des prestations B2B
- **Entrepreneur SaaS** qui communique en français
- **Responsable marketing** dans une PME francophone

### Ce n'est probablement PAS pour vous si :

- Vous ciblez un marché exclusivement anglophone US
- Vous travaillez dans un secteur ultra-régulé (banque, santé hospitalière) avec besoin de compliance sectorielle poussée
- Vous cherchez un outil no-code qui remplace totalement la réflexion stratégique
