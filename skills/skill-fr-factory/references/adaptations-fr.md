## 3. Typographie française

### 3.1 Règles strictes

| EN                                 | FR                                                                     |
|------------------------------------|------------------------------------------------------------------------|
| `"quotes"`                         | `« guillemets »` (avec espace insécable)                               |
| apostrophe `'` (droite)            | apostrophe typographique `'` (courbe)                                  |
| `...` trois points                 | `…` (un seul caractère U+2026)                                         |
| `-` tiret                          | `–` demi-cadratin (incises), `—` cadratin (dialogue, rupture)          |
| pas d'espace avant `: ; ? ! %`     | **espace insécable** (U+00A0) avant `:` `;` `?` `!` `%` `€` `°C`       |
| `1,000.00`                         | `1 000,00` (espace insécable milliers, virgule décimale)               |
| `$1,000`                           | `1 000 €` (symbole APRÈS, espace avant)                                |
| `MM/DD/YYYY` ou `01/15/2026`       | `JJ/MM/AAAA` ou `15/01/2026` — ou « 15 janvier 2026 » en littéraire    |
| `9:00 AM`                          | `9h00` ou `9 h 00` (typo stricte)                                      |
| `Mr.` / `Mrs.`                     | `M.` / `Mme`                                                           |
| `Dr.`                              | `Dr` (pas de point en FR)                                              |
| `vs.`                              | `c.` (contre) ou « vs » toléré                                         |

### 3.2 Majuscules accentuées (obligatoire en FR)

Les majuscules prennent l'accent : `À`, `É`, `È`, `Ê`, `Ç`.
> « À PARTIR DE » et non « A PARTIR DE ». Règle Académie française.

### 3.3 Siècles, nombres, pourcentages

- Siècles en petites capitales avec exposant : `xxi{e}` → **xxi{e} siècle**
- Ordinaux : `1er`, `1re`, `2e`, `3e` (pas `1st`, `2nd`)
- Pourcentage : `50 %` (avec espace insécable)
- Unités SI : `50 kg`, `20 °C`, `100 km/h` (espace insécable)

## 4. Culture business & ton

### 4.1 Ton

| Contexte US (EN)                   | Équivalent FR                                                          |
|------------------------------------|------------------------------------------------------------------------|
| Tutoiement direct dans le B2B      | **Vouvoiement par défaut**, tutoiement si contexte informel (SaaS jeune, agence créa) |
| Humour / punchlines assumés        | Plus sobre, éviter l'auto-congratulation, préférer la précision        |
| Superlatives ("amazing", "game-changer") | Éviter. Préférer : « efficace », « éprouvé », « documenté »        |
| CTA impératifs ("Get started now") | « Commencez », « Essayez », « Découvrez » — ton plus invitant          |
| Références à "hustle", "grind"     | À bannir. Préférer : « travail », « expertise », « rigueur »           |

### 4.2 Unités & formats régionaux

| US                                 | FR / Métropole UE                                                      |
|------------------------------------|------------------------------------------------------------------------|
| Miles / feet / pounds / °F / gal   | km / m / kg / °C / L (système métrique)                                |
| ZIP code                           | Code postal (5 chiffres en FR, 4 en BE, 4 en CH)                       |
| State (CA, NY…)                    | Département (75, 13, 69…) ou région (Île-de-France…)                   |
| SSN                                | Numéro de sécurité sociale (NIR, 13 chiffres + clé)                    |
| EIN / Tax ID                       | SIREN (9 chiffres), SIRET (14 chiffres), TVA intracomm. FR + 11 chiffres |
| Fiscal year ending June            | Exercice comptable calendaire (1er janv. → 31 déc.) majoritairement     |

### 4.3 Fêtes & saisonnalité

| US                                 | FR                                                                     |
|------------------------------------|------------------------------------------------------------------------|
| Thanksgiving, July 4th             | 14 juillet, 1er mai (fête du travail), 8 mai, 11 novembre              |
| Black Friday                       | Black Friday (OK FR, mais « Vendredi noir » littéral)                  |
| Spring / Summer / Fall / Winter    | printemps / été / automne / hiver                                      |
| Back-to-school (fin août)          | **Rentrée** (début septembre) — moment commercial MAJEUR en FR         |
| Fiscal year close                  | **Clôture fiscale** (mars/avril pour IR, variable pour IS)             |

### 4.4 Prénoms & exemples

Éviter John/Jane, utiliser : Marie, Pierre, Julien, Camille, Sophie, Thomas, Lucie, Antoine, Sarah, Karim, Aïssa, Mehdi, Chloé, Nicolas.

Éviter Springfield/Anytown, utiliser : Paris, Lyon, Marseille, Toulouse, Nantes, Bordeaux, Lille, Strasbourg — et pour la ruralité : Quimper, Limoges, Clermont-Ferrand, Besançon.
