Questions fréquentes

Tout ce que vous voulez savoir avant d'acheter.

Est-ce que je peux l'utiliser pour plusieurs clients ?
Oui, sans limite. Une fois le pack acheté, vous pouvez l'utiliser pour votre propre activité et pour accompagner vos clients sans frais supplémentaires. La seule restriction : vous ne pouvez pas revendre ou redistribuer le pack.
Il me faut un abonnement Claude pour l'utiliser ?
Oui, le pack fonctionne avec Claude (Pro, Team, Enterprise ou API). Vous pouvez l'utiliser via Claude Code (recommandé), Claude.ai avec les Projects, ou via l'API. Pas d'abonnement supplémentaire nécessaire au-delà de votre abonnement Claude.
Comment je reçois le pack après achat ?
Vous recevez immédiatement un lien de téléchargement (fichier .zip) contenant tous les skills, templates et références. Un email avec un guide d'installation pas à pas en français vous est également envoyé. L'installation prend moins de 5 minutes.
Est-ce compatible avec OpenClaw / Claude Code ?
Oui, 100 % compatible avec Claude Code et OpenClaw. Le pack utilise le format standard SKILL.md, donc il fonctionne aussi avec Codex CLI, Cursor, ChatGPT, GitHub Copilot, Gemini CLI et 14+ autres assistants IA compatibles avec le standard Agent Skills ouvert.
Je n'utilise pas Claude, seulement ChatGPT. Ça marche quand même ?
Oui. Le standard SKILL.md est ouvert et compatible avec ChatGPT (via GPTs ou Projects), Cursor, Copilot, et tous les assistants IA modernes. Les instructions sont rédigées en langage naturel, elles fonctionnent avec n'importe quel modèle suffisamment capable (GPT-4, Claude Opus, Gemini Pro, etc.).
Je ne suis pas technique, je peux l'utiliser ?
Oui. Si vous savez utiliser Claude.ai (avec les Projects), vous savez utiliser ce pack. Un guide d'installation pas à pas est fourni, avec captures d'écran. Vous pouvez aussi nous contacter pour une aide ponctuelle.
Les mises à jour sont-elles incluses ?
Oui pour les mises à jour mineures (corrections, ajout de templates, nouveaux exemples) qui sont incluses à vie. Les versions majeures (v2, v3) seront proposées à -50 % pour les clients existants.
Puis-je demander un remboursement si ça ne me convient pas ?
Absolument. Vous avez 14 jours pour tester le pack. Si vous n'êtes pas satisfait, un email suffit pour obtenir un remboursement intégral, sans justification. Aucun risque pour vous.
Est-ce adapté à mon marché (Maurice, Afrique FR) ?
Oui, le pack a été pensé pour toute la zone francophone : France métropolitaine, Belgique, Suisse romande, Maurice et Afrique francophone. Les spécificités locales (tarification, fiscalité, culture) sont documentées dans les références incluses.
